# TimERP Design System

## Brand Direction

- Style: High-tech, professional, asymmetric, data-command
- Tone: Precise, confident, direct
- Visual principle: strong contrast, layered surfaces, clean data blocks

## Typography

- Sans: `Exo 2`
- Mono: `Space Mono`
- Serif: `Lora`
- Heading weight: `600-700`
- Body weight: `400-500`
- Body line-height: `1.55-1.7`

## Spacing And Shape

- Base spacing unit: `4px`
- Main section gap: `48-72px`
- Card radius: `16px-24px`
- CTA radius: `10px-14px`
- Border width: `1px`

## Shadows

- Sm: `0 2px 8px rgba(2, 6, 23, 0.06)`
- Md: `0 8px 24px rgba(2, 6, 23, 0.10)`
- Lg: `0 20px 56px rgba(2, 6, 23, 0.14)`

## Motion

- Duration: `200ms`
- Curve: `cubic-bezier(0.22, 1, 0.36, 1)`
- Hover: color/border/glow only
- No scale-jump on cards

## Components Rules

- Navbar: floating, top spacing, blur + subtle border
- Hero: asymmetric split, one strong KPI strip
- Cards: layered background + clear border
- CTA: one primary, one secondary
- Icons: Lucide only, no emoji
- Grid texture: always present, low opacity

## Accessibility Rules

- Minimum text contrast: WCAG AA
- Keyboard focus ring always visible
- Motion should respect reduced-motion
- Do not use color as only status cue
